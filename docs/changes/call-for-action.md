# Call for Action

This page holds all the Call for Actions in case of breaking changes. 

## v1.13.3 - Apprise Server URL

⚠️ We no longer add /notify to the url, this will be an breaking change. Existing Apprise users need to adjust their URLS.

??? danger "How to change it"
    1. Open the Notification Settings 
    2. Go the Apprise tab
    3. Under Apprise Server URL 
    4. Add `/notify` to the URL
    5. Save the Settings
