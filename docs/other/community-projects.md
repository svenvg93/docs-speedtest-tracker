# Community Projects

This page lists community projects that use SpeedTest Tracker as a base for their own projects. If you have a project that you would like to share, please submit a pull request to add it to this list.

## Projects

* [Kubernetes Manifests] - Community-maintained Kubernetes manifests for deploying Speedtest Tracker in a Kubernetes cluster.
* [SpeedtestTrackerBot] â€” SpeedtestTrackerBot is a lightweight Discord bot that integrates with the Speedtest Tracker API to provide real-time network performance data via slash commands like `/latest`, `/stats`, `/result`, and `/run`. It's easy to set up with Docker or systemd.
* [Speedtest Tracker App (iOS)] - Speedtest Tracker App is a native iOS client for Speedtest Tracker, which integrates with its API to allow you to perform actions or see stats straight from your phone.
* Stream Deck - Use the API to show the most recent result on your Stream Deck. Shoutout to [MartynKeigher] on GitHub for [providing] these instructions.
